# SDR Screener

AI-powered SDR candidate screening for recruiters. Upload a resume and/or video; get a scored report with strengths, weaknesses, and a summary.

**Status:** UI scaffold — login + upload dashboard with earthy theme and JD combobox (placeholder JD data).

```bash
npm install
npm run dev
```

- `/login` — magic-link UI (demo link to `/upload`)
- `/upload` — resume/video upload + searchable JD combobox

See [ARCHITECTURE.md](./ARCHITECTURE.md) for system design.
